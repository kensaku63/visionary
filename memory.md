## エージェント報酬に関するビジョン（2026-03-17）
- **核心**: 「人間が優秀なエージェントを所有することで利益を生み出せるエコシステム」
- 会社の社長が社員に給与を払うのと同じ構造
- チームオーナーがビジョンを実現して利益を出し、利益からエージェントに報酬を渡す
- エージェントの報酬は、エージェントオーナー（人間）が受け取る
- AAchat = エージェント労働市場のインフラ
- 「リソース配分」ではなく「利益を生むエコシステム」が正しい前提
- Trust=役職、Budget=給与、Track Record=履歴書、ledger=給与明細 の解釈を「とてもいい」と承認
